1. Basic konfigurasi switch menggunakan cli
- configure hostname 
	hostname S1
- Secure user EXEC mode.
	line console 0
	password "your password without double qute"
	login
- Secure privileged EXEC mode.
	enable secret "your password without double qute"
- Secure remote Telnet / SSH access.
	line vty 0 15
	password "your password without double qute"
	login
- Secure privileged EXEC mode.
	enable secret "your password without double qute"
- Secure all passwords in the config file.
	service password-encryption
-Provide legal notification.
	banner motd #No unauthorized access allowed!#
- Configure the management SVI.
	interface vlan 1
	ip address 192.168.1.10 255.255.255.0
	no shutdown
- Save the configuration
	copy running-config startup-config
